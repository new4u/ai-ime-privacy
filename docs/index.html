AI-IME 隐私政策

1. 信息收集

AI-IME 扩展程序会收集以下信息：

  • 您输入的文字内容（仅用于实时翻译和处理）
  • Notion 登录状态（如果您选择使用 Notion 集成功能）
  • 用户电子邮件地址（仅用于验证订阅状态）

2. 信息使用

我们收集的信息将用于：

  • 提供实时输入法服务
  • 同步内容到您的 Notion 账户（如果您启用此功能）
  • 验证您的订阅状态
  • 改进我们的服务质量

3. 信息存储

所有用户数据都存储在您的本地浏览器中，使用 Chrome 存储 API 进行安全存储。我们不会在服务器上永久存储您的个人信息。

4. 第三方服务

我们使用以下第三方服务：

  • Notion API（用于笔记同步，仅在您授权后使用）
  • Buy Me a Coffee（用于订阅验证）

5. 数据安全

我们采取以下措施保护您的数据：

  • 所有数据传输使用 HTTPS 加密
  • API 密钥和认证信息使用安全存储
  • 定期检查和更新安全措施

6. 用户权利

您有权：

  • 随时查看和删除您的数据
  • 选择是否使用 Notion 集成功能
  • 要求我们提供您的数据副本

7. 隐私政策更新

我们可能会不时更新此隐私政策。更新后的政策将在本页面公布，并在扩展程序更新时通知用户。
